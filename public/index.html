String Pool & Equality Java

Understanding == vs .equals() is fundamental.

String s1 = "Hello"; // String Pool
String s2 = new String("Hello"); // Heap

System.out.println(s1 == s2);      // false (Ref check)
System.out.println(s1.equals(s2)); // true (Value check)
Tip: Always use .equals() for content comparison.

Vector Reallocation C++

Prevent expensive copies by reserving memory.

std::vector v;
v.reserve(1000); // Allocate memory once

for(int i=0; i<1000; i++) {
    v.push_back(i); // No reallocation needed
}

Event Loop & Scope JS

    setTimeout(() => console.log(i), 100);
