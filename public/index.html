Mutable Default Arguments Python

Avoid using mutable objects like lists as default arguments.

# ‚ùå WRONG: list is shared across calls
def add(item, target=[]):
    target.append(item)
    return target

# ‚úÖ CORRECT: Use None as sentinel
def add(item, target=None):
    if target is None:
        target = []
    target.append(item)
    return target
Insight: Default values are evaluated once at definition time.

String Pool & Equality Java

Understanding == vs .equals() is fundamental.

String s1 = "Hello"; // String Pool
String s2 = new String("Hello"); // Heap

System.out.println(s1 == s2);      // false (Ref check)
System.out.println(s1.equals(s2)); // true (Value check)
Tip: Always use .equals() for content comparison.

Vector Reallocation C++

Prevent expensive copies by reserving memory.

std::vector v;
v.reserve(1000); // Allocate memory once

for(int i=0; i<1000; i++) {
    v.push_back(i); // No reallocation needed
}

Event Loop & Scope JS

Use let for block scoping in loops.

// ‚úÖ "let" creates a new binding each iteration
for (let i = 0; i < 3; i++) {
    setTimeout(() => console.log(i), 100);
}
// Output: 0, 1, 2
